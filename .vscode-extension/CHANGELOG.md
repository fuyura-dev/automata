# Change Log

All notable changes to the "transformation-rules-syntax" extension will be documented in this file.

## [0.0.1] - 2025-12-22

### Added
- Initial release
- Syntax highlighting for `.rules` files
- Support for variables (C, V, R)
- Support for affixes (quoted strings)
- Support for arrow operators (->)
- Support for JSON metadata
- Support for line comments (//)
